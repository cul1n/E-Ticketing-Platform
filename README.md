# E-Ticketing-Platform
Project made for the Advanced Object-Oriented Programming Course @ FMI UNIBUC.

Functionalities:
- create new user
- log in

For Users:
- view account details
- add funds to their balance
- view owned tickets
- view ticket details
- filter tickets by location
- buy tickets

For Admin:
- view user list
- view audit
- create events
- view events
- create tickets
- view tickets
